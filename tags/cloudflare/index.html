Cloudflare Zerotrust 使用API删除Access

# 个人用户可能用到的ZeroTrust功能

1.Tunnel:免费内网穿透(需要域名)
2.Access:保护内网穿透的NAS和/自用面板/管理后台等(免费但需绑定支付方式)
3.ZeroTrust VPN(被墙了)

Access很好用,没有通过Cloudflare验证的流量根本不会发到自己的服务器上,也许是最安全的鉴权方式.

Access不绑定支付方式也能添加,但在添加之后在网页上没法修改/删除,会导致自己被锁在网页外面.

以下是用Cloudflare API删除Access的方法

# Linux Curl调用Cloudflare API

Cloudflare Api Document

列出所有Application

curl https://api.cloudflare.com/client/v4/accounts/$CloudflareAccountID/access/apps -H "X-Auth-Email: $Cloudflare Email" -H "X-Auth-Key: $CloudflareApi Key"

删除指定Application

curl https://api.cloudflare.com/client/v4/accounts/$CloudflareAccountID/access/apps/$AppID -X DELETE -H "X-Auth-Email: $Cloudflare Email" -H "X-Auth-Key: $Cloudflare Api Key"

清除bash记录

history -c
rm ~/.bash_history

#

理论上用Cloudflare API也可以在没有绑定支付方式的情况下添加/修改Access,达到正常使用的目的.
但不推荐这样做,如果API突然增加限制就会彻底没法修改了.(没付钱Cloudflare人工客服肯定不会理你,删除账号同一个邮箱不能创建新的).
