## 已知问题

# OpenWRT 25.12.0-rc1已知问题

  • 5Ghz Wifi,信道选auto无法启动,需要手动设置信道为36
  • 5Ghz Wifi即使开启160mhz也无法协商到超过1200mbps,测速跑不满千兆,对无线速度有要求就再等等
  • 虽然路由器有128mb闪存,但OpenWRT可用容量仅60mb
  • 内存占用异常:不装任何插件内存占用300mb+(正常情况下512mb内存应该占用200mb左右)
  • 以上问题rc2一个都没有修复

# 路由器自身的问题

  • 这款路由器三个千兆网口共享1G带宽,无线/有线都基本不可能跑满2.5G,不建议为了能刷机购买.见acwifi评测
  • IPQ5018处理器性能较弱(双核1GHZ A53,相当于NanoPi R2s软路由的一半不到),且原版OpenWRT不支持硬件加速,不建议用于拨号/翻墙

# 评价

Xiaomi AX6000本来就是一款除了无线性能勉强及格其它一无是处的路由器,刷OpenWRT后无线也不及格了,不建议刷

## 测速

  • 开启Software flow offloading
  • 内网使用OpenSpeedTest
  • 外网使用中科大测速站
  • wan口=2.5G网口
测试场景速度 (下载/上传)备注

测了很多遍,确实是这种奇怪的结果,我不知道原因

5Ghz Wifi目前肯定工作不正常,但只要固件不支持Qualcommax NSS(高通硬件加速),这么烂的CPU能不能跑满5Ghz Wifi或千兆网络依然是问题.

## 刷机步骤

# 0.降级到1.0.55 (Optional)

有些新生产的Xiaomi AX6000路由器无法降级到低版本,可能是因为硬件规格改变(缩水)但没有更换型号,可以先在当前版本尝试解锁ssh,失败再降级

Xiaomi AX6000历史固件下载

实测我自己的路由器可以从最新稳定版(1.0.122)直接降级到1.0.55

# 1.使用xmir-patcher解锁ssh

  • 网上搜到的教程解锁ssh需要装OpenWRT虚拟机开无线热点,现在xmir-patcher开箱即用,不需要管理员权限,可以自动获取sn/计算root密码/开启ssh/备份固件/刷机…

下载并解压源码.zip,点击run.bat


==========================================================

Xiaomi MiR Patcher


 1 - Set IP-address (current value: 192.168.31.1)
 2 - Connect to device (install exploit)
 3 - Read full device info
 4 - Create full backup
 5 - Install EN/RU languages
 6 - Install permanent SSH
 7 - Install firmware (from directory "firmware")
 8 - {{{ Other functions }}}
 9 - [[ Reboot device ]]
 0 - Exit

Select:

选2 解锁ssh

# 2.备份

选4 备份闪存内容,刷回原版有用,但变砖可能要拆机救
选6 固化ssh

# 3.下载Openwrt固件

openwrt firmware-selector 选择Xiaomi AX6000, 25.12.x

下载第一个SYSTEMUPGRADE (xxx-squashfs-sysupgrade.bin)
和第三个INITRAMFS-FACTORY (xxx-initramfs-factory.ubi)

# 4.刷机

将...-initramfs-factory.ubi放入./xmir-patcher-main/firmware

选7 刷机

  • OpenWRT默认不开启WIFI,必须连网线
  • OpenWRT默认ip是192.168.1.1,首次开机不要接上游避免ip冲突

重启连接网线到lan口(三个千兆口的任意一个),浏览器输入192.168.1.1即可进入luci界面,没有密码.

在System -> Backup / Flash Firmware -> Flash Image
刷入...-squashfs-sysupgrade.bin

重启
